## What It Does

n8n-MCP gives Claude and other AI assistants deep knowledge about n8n's workflow automation platform:

- **2,063 n8n nodes** — 816 core + 1,247 community nodes
- **99% property coverage** with detailed schemas
- **87% documentation coverage** from official n8n docs
- **265 AI tool variants** with full documentation
- **156 ranked example configurations** from popular templates
- **2,352 workflow templates** with AI metadata

## Safety Warning

⚠️ **NEVER edit production workflows directly with AI!** Always test in development first.

## Quick Start

**Hosted (free tier, 100 calls/day):**
Visit [dashboard.n8n-mcp.com](https://dashboard.n8n-mcp.com)

**Self-hosted:**
```bash
npx n8n-mcp
```

**Docker:**
```bash
docker run ghcr.io/czlonkowski/n8n-mcp
```
