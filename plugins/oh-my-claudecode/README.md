# oh-my-claudecode

> Multi-agent orchestration for Claude Code. Zero learning curve.

[![npm version](https://img.shields.io/npm/v/oh-my-claude-sisyphus?color=cb3837)](https://www.npmjs.com/package/oh-my-claude-sisyphus)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**Don't learn Claude Code. Just use OMC.**

oh-my-claudecode provides a complete teams-first multi-agent orchestration layer on top of Claude Code. Install once, then drive complex multi-agent workflows through simple CLI commands.

## Core Features

- **Multi-agent orchestration** — parallel Claude Code instances collaborating on tasks
- **Zero config** — install via plugin marketplace, immediately ready
- **Skills & Hooks** — pre-built skill library + hook system for lifecycle automation
- **CLI reference** — `omc` command with full subcommand suite
- **Team workflows** — specialized agent roles with handoff protocols
- **MCP server bridge** — exposes orchestration as MCP tools for other agents

## Install

```bash
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
```

Or via npm:
```bash
npm install -g oh-my-claude-sisyphus
```
