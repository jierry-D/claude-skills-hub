# Marketing Skills for Claude Code

> Source: https://github.com/coreyhaines31/marketingskills

CRO, copywriting, SEO, analytics, and growth engineering skills for Claude Code. 35,312 ⭐

## Skills Included (45+)

- **Conversion Optimization**: CRO audits, signup flows, onboarding, paywalls, popups
- **Content & Copy**: Copywriting, email sequences, social content, video scripts
- **SEO & Discovery**: Technical audits, AI search (GEO/AEO), programmatic SEO, schema
- **Paid Marketing**: Ad campaigns, creative generation (Google, Meta, LinkedIn, TikTok)
- **Analytics & Testing**: A/B testing, measurement tracking, analytics setup
- **Retention**: Churn prevention, customer health scoring
- **Growth**: Referral programs, free tools, co-marketing partnerships
- **Strategy**: Pricing analysis, launch playbooks, competitive intelligence
- **Sales**: RevOps, sales enablement, prospecting sequences

## Foundation

All skills reference a `product-marketing` base skill for consistent positioning.

## Installation

```bash
npx skills add coreyhaines31/marketingskills
# or via Claude Code plugin:
/plugin marketplace add coreyhaines31/marketingskills
```
