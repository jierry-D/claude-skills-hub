# UI UX Pro Max — Design Intelligence Skill

> Source: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill — 97,170 ⭐

## Overview

An AI-powered design intelligence skill for building professional user interfaces across multiple platforms and frameworks. Analyzes project requirements and produces tailored design systems automatically.

## What's Included

- **67 UI Styles**: Glassmorphism, Claymorphism, Minimalism, Brutalism, AI-Native UI, and 62 more
- **161 Color Palettes**: Aligned with specific product categories
- **57 Font Pairings**: Curated from Google Fonts
- **161 Industry-Specific Reasoning Rules**: SaaS, Finance, Healthcare, E-commerce, Creative, and more
- **22 Tech Stack Support**: React, Next.js, Vue, Flutter, SwiftUI, and more
- **99 UX Guidelines**: Best practices and accessibility standards

## Usage

Skill activates automatically for UI/UX requests in Claude Code, Cursor, Windsurf, and other supported platforms. Simply describe your design needs naturally.

For direct access:
```
"Generate design system with ASCII output"
```

## Installation

```bash
# Via Claude Marketplace
/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill

# Via CLI (recommended)
npm install -g ui-ux-pro-max-cli
uipro init --ai claude
```

Requires Python 3.x for search functionality. MIT licensed.
