# Humanizer

> Source: https://github.com/blader/humanizer

Claude Code skill that removes signs of AI-generated writing from text. 26,600 ⭐

## What it detects and rewrites

33 AI writing pattern categories:
- **Content inflation**: Undue significance, notability framing, broader-trend filler
- **Shallow analysis**: Present-participle overuse ("-ing" constructions)
- **Promotional tone**: "vibrant," "nestled," "breathtaking," "transformative"
- **Vague sourcing**: "experts argue," "studies show" without citations
- **Vocabulary tells**: "crucial," "landscape," "pivot," "interplay," "delve"
- **Grammar substitution**: "serves as" instead of "is," "boasts" instead of "has"
- **Punctuation markers**: Excessive em dashes (hard constraint: remove all)
- **Hedging & filler**: Over-qualification, generic positive conclusions

## Voice Matching

Provide a writing sample to calibrate rewrites to your personal style (rhythm, word choice, punctuation habits).

## Usage

Install as a Claude Code skill, then invoke when reviewing or editing any text output.
