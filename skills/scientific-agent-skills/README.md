# Scientific Agent Skills

> Source: https://github.com/K-Dense-AI/scientific-agent-skills

Turn any AI agent into an AI Scientist. 147 ready-to-use skills + 100+ scientific databases. 29,521 ⭐

## Coverage

- **Biology**: Bioinformatics, genomics, proteomics (UniProt, NCBI, STRING)
- **Chemistry**: Cheminformatics, drug discovery (PubChem, ChEMBL, RDKit)
- **Medicine**: Clinical research, FDA data (ClinicalTrials.gov, FDA, DepMap)
- **ML/Data Science**: PyTorch Lightning, Scanpy, scikit-learn optimized workflows
- **Lab Automation**: Benchling, DNAnexus, Opentrons, LatchBio
- **Publishing**: PubMed, arXiv, bioRxiv, Semantic Scholar

npx skills add K-Dense-AI/scientific-agent-skills
