# Caveman — Token-Efficient AI Agent Skill

> Source: https://github.com/JuliusBrussee/caveman — 77,438 ⭐

> "why use many token when few do trick"

## What It Does

Teaches your AI agent to communicate concisely — preserving all technical accuracy while compressing conversational overhead by ~65%.

## Compression Levels

| Level | Token Reduction | Use Case |
|-------|-----------------|----------|
| lite | ~25% | Light compression |
| full | ~65% | Daily development |
| ultra | ~80% | Maximum savings |
| wenyan | ~87% | Classical Chinese compression |

## Commands

- `/caveman` — Toggle compression level
- `/caveman-commit` — Concise commit messages
- `/caveman-review` — One-line PR feedback
- `/caveman-compress` — Rewrite memory files (~46% input token savings)
- `/caveman-stats` — Real-time usage statistics

## Installation

Single-line install (macOS/Linux/Windows/WSL):
```bash
curl -fsSL https://github.com/JuliusBrussee/caveman/raw/main/install.sh | bash
```

Requires Node ≥18. Compatible with 30+ agents including Claude Code, Cursor, Windsurf, Cline, and Copilot.

## Why It Works

Research cited in the README suggests constraining large models to brief responses improved accuracy by 26 points on certain benchmarks — verbosity doesn't guarantee better outputs. Code, commands, and error strings are preserved exactly; only conversational style is compressed.
