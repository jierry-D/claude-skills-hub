- **功能简介**: 专为 Apple Silicon Mac 设计的 vLLM 风格推理服务器。同时暴露 OpenAI `/v1/*` 和 Anthropic `/v1/messages` 两套 API，支持连续批处理、分页 KV 缓存和 SSD 分层缓存。与 Claude Code 完全兼容，可用本地模型替代 Claude API 进行测试。
